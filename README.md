# 🌿 Proyecto Destacado — Herencia Verde

### 🔗 Demo del Proyecto
👉 https://herencia-verde.netlify.app

Herencia Verde es una plataforma digital enfocada en sostenibilidad y conciencia ambiental, diseñada para conectar tecnología, comunicación visual y experiencia de usuario en una solución moderna e interactiva.

El proyecto fue desarrollado con un enfoque responsive y visualmente atractivo, buscando transmitir impacto ambiental mediante herramientas digitales, diseño moderno y navegación intuitiva.
